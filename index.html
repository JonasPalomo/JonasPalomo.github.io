Questboard

Questboard is a website that was developed using Python and Django framework. It utilizes postgreSQL database in order to store relevant information needed for the website. It is designed to be a task distributor used by teachers in order to disseminate tasks to their students. A teacher can create a Questboardfor a specific topic in the course, and assign the number of stars needed to get full marks on the Questboard. The teacher can then post quests in the Questboard where students can participate and earn stars for the questboard.

COVID-19 Dashboard

My coursemates and I worked on this dashboard project for our Data Visualization course. The intended audience of this dashboard is directed towards the general health sector and governments of each country in order to give useful recommendations that are grounded on data for the sake of the safety of their citizens. The dataset contains thousands of rows of data, which includes confirmed cases, recoveries and deaths. It also provided data on world vaccination progress, which we used to see the correlation between vaccination progress vs. GDP.
