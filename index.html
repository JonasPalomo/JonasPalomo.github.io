Questboard

Questboard is a website that was developed using Python and Django framework. It utilizes postgreSQL database in order to store relevant information needed for the website. It is designed to be a task distributor used by teachers in order to disseminate tasks to their students. A teacher can create a Questboardfor a specific topic in the course, and assign the number of stars needed to get full marks on the Questboard. The teacher can then post quests in the Questboard where students can participate and earn stars for the questboard.

COVID-19 Dashboard

Using Tableau Desktop and Tableau Prep, I created a dashboard that explains the current progress of world vaccination and COVID-19 updates of each country. It includes interactive graphs and charts that a user can adjust to see the numbers change with respect to the month and year. The dataset contains hundreds of thousands of rows of data, which includes confirmed cases, recoveries and deaths. The intended audience of this dashboard is directed towards the general health sector and governments of each country in order to give useful recommendations that are grounded on data

Simulation of COVID-19 w/ Vaccine Rollout

I created this simulation modelled after the SIVR model using Python. I also used Python libraries such as NumPy and pandas for data manipulation, Matplotlib for data visualization, and AgentPy framework to create the data model and simulation itself. The model simulates the behavior of a population being infected by COVID-19 and the effect of a vaccine rollout after a specific time-step.
